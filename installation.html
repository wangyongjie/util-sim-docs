# 快速上手

# NPM

推荐使用 npm 的方式安装，它能更好地和 webpack (opens new window) 打包工具配合使用。

npm i util-sim -S

# 导入

import * as util from 'util-sim' 

或

const utils = require('util-sim')

# 使用方式

例：校验是否是手机号

let phone = '13579876263'
utils.validate.isPhone(phone)

例：获取当前时间

utils.common.getNowDate()
